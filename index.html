Installation

Initial installation:

Prerequisites: Make, awk, cut and xargs all of which are available via http://www.cygwin.com for Windows, other platforms often have these.

:  $ cd ~
:  $ git clone git://github.com/developernotes/emacs-setup.git .emacs.d
:  $ cd .emacs.d
:  $ make install

Updating submodules:

:  $ cd ~/.emacs.d
:  $ make submodules-pull

Syncing submodules to current ref.

:  $ cd ~/.emacs.d
:  $ make submodules-update
